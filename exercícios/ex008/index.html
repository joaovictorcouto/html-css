Citações completas

Segundo Jeff Noble, no seu livro HTML para Leigos:

A diferença entre elementos inline e um bloco de texto é importante. Os elementos HTML neste capítulo descrevem os blocos de texto. Eles podem conter outros elementos de bloco, bem como elementos inline. Os elementos de bloco são usados para estruturar o conteúdo da página. Eles podem ser usados para agrupar outros elementos, como parágrafos, listas e cabeçalhos. Alguns elementos de bloco comuns incluem <div>, <p>, <h1> a <h6>, <ul>, <ol> e <li>.
